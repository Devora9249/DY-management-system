# Nonprofit Management System

A comprehensive web application for managing non-profit organizations, including donations, donors, salaries, and financial reports.

## Overview
The system was developed as a collaborative personal initiative, designed to support the daily operational and financial management of a non-profit organization.

The system focuses on clear business logic, data integrity, and a scalable architecture.

## Tech Stack
### Frontend
- React
- JavaScript
- HTML, CSS

### Backend
- Node.js
- Express
- RESTful API

### Database
- MongoDB

## Features
- Donation management (standing and one-time)
- Donor management
- Salary and expense tracking and debts
- Income and expense reports
- Server-side business logic implementation
- Automated processes such as sending emails and handling data
- RESTful client-server communication

## Architecture
- Separation of Frontend and Backend
- REST-based architecture
- Centralized business logic
- Clean and maintainable code structure
